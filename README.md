# 🚀 UniConnect - Hack To The Future (HTTF)

## 🎯 HackByIfri 2026

**Thème : Intégration efficace du numérique dans l'apprentissage**

UniConnect est une plateforme web conçue pour aider les étudiants,
notamment ceux en double inscription, à optimiser leur gestion
académique : emplois du temps, cours, évaluations et organisation du
temps.

## ✨ Fonctionnalités Principales

### 🎓 Espace Étudiant

-   Tableau de bord complet : actualités, épreuves et cours.
-   Programme hebdomadaire : planning détaillé.
-   Personnalisation : gestion du profil et mode sombre/clair.
-   Options Premium : corrections d'épreuves, quiz par matière et outils
    d'aide à la révision.

### 🏫 Espace Entité (École / Faculté)

-   Gestion pédagogique : ajout et affichage des cours, épreuves et
    programmes.
-   Communication : publication d'actualités.
-   Analyses : statistiques sur les activités de l'entité.

### 🛡️ Espace Administrateur

-   Sécurité : validation des entités via liens d'inscription uniques et
    sécurisés.
-   Supervision : statistiques globales des universités et entités.

------------------------------------------------------------------------

## 🛠️ Stack Technique

-   Frontend : HTML5, CSS3 (Responsive Design), JavaScript.
-   Backend : PHP.
-   Base de données : MySQL.
-   Design : Stitch with Google.

------------------------------------------------------------------------

## 💰 Modèle Économique

-   Version gratuite : gestion d'un seul établissement.
-   Abonnement (500 FCFA / mois) :
    -   Gestion multi-établissements
    -   Accès aux quiz IA
    -   Corrections d'épreuves
    -   Outils de révision
